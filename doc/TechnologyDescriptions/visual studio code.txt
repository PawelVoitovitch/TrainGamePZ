W ramach naszej pracy planujemy również korzystać z pomocnicznych narzędzi programistycznych, w tym z edytora tekstu Visual Studio Code. Edytor ten jest bardzo łatwo konfigurowalny poprzez doinstalowywanie wszelkich rozszerzeń dla różnych języków. Będzie on nam głównie służył do pomocy przy tworzeniu kodu po stronie front-end we frameworku React Js. Jego zaletami są m.in. łatwość konfiguracji, lekkość - nie pożera wielu zasobów komputera oraz jest całkowicie darmowy.

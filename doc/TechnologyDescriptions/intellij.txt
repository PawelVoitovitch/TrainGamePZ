IntelliJ to zintegrowane środowisko programistyczne którego zdecydowaliśmy się używać jako główne środowisko w trakcie projektowania i programowania naszej aplikacji. Jest ono stworzone przez firmę JetBeans i postrzegane jako jedno z najlepszych narzędzi programistycznych do projektowania aplikacji. Wyposażone w różne narzędzia programistyczne zdecydowanie pomoże nam i zwiększy jakość i szybkość tworzonego oprogramowania. Posiada szerokie wsparcie dla technologii w których będziemy tworzyć naszą aplikację, takich jak Java Spring Boot oraz React Js. Do tego posiada zintegrowane wsparcie systemu kontroli wersji Git oraz wsparcie dla projektowania baz danych SQL. Niewątpliwie największą zaletą jest również darmowa licencja wersji Ultimate dla studentów oraz nauczycieli w celach edukacyjnych z której planujemy skorzystać.
